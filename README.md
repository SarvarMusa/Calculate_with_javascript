# JavaScript Calculator

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Description: A sleek, interactive calculator built with vanilla JavaScript, HTML, and CSS. It solves the everyday problem of quick arithmetic calculations directly in the browser, featuring a smooth dark/light mode toggle powered by a 3D tilt effect using vanilla-tilt.js.

## Tech Stack
- **Languages:** JavaScript (ES6), HTML5, CSS3
- **Libraries:** vanilla-tilt.js, Font Awesome 5
- **Platform:** Browser (no build tools or dependencies required)

## Setup instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/SarvarMusa/Calculate_with_javascript.git
   cd Calculate_with_javascript
   ```

2. Open `html/index.html` directly in your browser — no server or install needed:
   ```bash
   open html/index.html   # macOS
   xdg-open html/index.html  # Linux
   ```

## Environment variables
No environment variables required. This is a purely frontend project.

## Usage examples
- Click number buttons to enter digits
- Use `+`, `-`, `*`, `/` for arithmetic operations
- Press `=` to evaluate the expression
- Press `C` to clear the screen
- Toggle the **🌙 / ☀️** switch in the top corner to switch between dark and light mode
